## Java 코드의 구조
* java 의 소스 코드는 `*.java`라는 이름으로 Text 형식의 파일을 작성한다.
* java 의 소스 코드는 `class *** {}` 코드 블록으로 시작한다. 그래서 java 의 소스 코드를 클래스 파일이라고 부른다.
